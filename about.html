About PixelVision

PixelVision is a learning project that demonstrates how image formats affect quality, transparency, and file size through clear, interactive examples.

Purpose

Help learners pick the right format for photos, graphics, and UI assets by showing practical, side‑by‑side comparisons.

Technologies

  • HTML5 and modern CSS
  • Vanilla JavaScript for the comparison slider
  • Optimized assets in JPEG, PNG, WEBP, and AVIF

How to Use

Open the Comparison page, drag the handle to reveal differences, and switch formats using the dropdowns. File sizes update live for quick inspection.
